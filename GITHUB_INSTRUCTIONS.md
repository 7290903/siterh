# Инструкция по отправке файлов на GitHub

## Быстрый способ (через скрипт)

1. Запустите файл `push-to-github.bat` двойным кликом
2. Следуйте инструкциям в консоли
3. При первом запуске введите URL вашего GitHub репозитория

## Ручной способ

### 1. Проверьте статус репозитория
```bash
git status
```

### 2. Добавьте все файлы
```bash
git add .
```

### 3. Создайте коммит
```bash
git commit -m "Ваше сообщение коммита"
```

### 4. Если репозиторий еще не создан на GitHub:
   - Создайте новый репозиторий на GitHub.com
   - Скопируйте URL репозитория

### 5. Добавьте remote (если еще не добавлен)
```bash
git remote add origin https://github.com/ВАШ_USERNAME/ВАШ_РЕПОЗИТОРИЙ.git
```

### 6. Отправьте файлы на GitHub
```bash
git push -u origin main
```
или
```bash
git push -u origin master
```
(в зависимости от названия вашей основной ветки)

## Если возникли проблемы

- **Ошибка аутентификации**: Используйте Personal Access Token вместо пароля
- **Ветка не найдена**: Проверьте название ветки командой `git branch`
- **Конфликты**: Сначала выполните `git pull` для синхронизации
